## ✨ Core Features

- **View Bot** — Real watch-time, partial/full views, retention simulation
- **Like / Dislike** — Mass liking or disliking videos
- **Comment Bot** — Randomized smart comments (or custom list), replies
- **Subscriber Bot** — Auto subscribe/unsubscribe
- **Search Boost** — Rank videos higher in search results
- **Playlist & Share** — Add to playlists, share links
- **Multi-account** — Use multiple Google/YouTube accounts
- **Proxy Support** — HTTP • SOCKS5 • Residential / Mobile (highly recommended)
- **Browser Fingerprint** — Chrome/Firefox emulation + canvas/WebGL spoofing
- **Human Behavior** — Randomized watch duration, mouse movements, scroll, pauses
- **Threads / Bots** — Run hundreds of parallel sessions
- **Task Scheduler** — Set daily/hourly loops, ramp-up mode
- **Live Stats** — Views added • Likes • Comments posted • Success rate • CPM simulation
- **Igyx Dark UI** — Animated progress, real-time logs, proxy status

## 🚀 Quick Start

1. Launch YouTubeBot
2. Enter target **Video URL** or **Channel URL**
3. Select action(s):

   - **Views** — Main mode for watch-time
   - **Likes** / **Comments** / **Subs**
   - **Search Rank** (keywords to boost)

4. Configure:

   - **Proxy** — Load list (residential best)
   - **Threads/Bots** — 10–200 (start low)
   - **Duration** — Watch time per view (30s–full)
   - **Comments** — Load list or enable random/smart
   - **Delay** — Between actions (3–15s realistic)

5. Hit **Start** — monitor live dashboard

## 🛡️ Pro Tips for Longevity & Success

- **Residential/mobile proxies** only — datacenter = instant shadowban
- Rotate **fingerprints** every 5–20 sessions
- Keep **watch duration** realistic (mix short + long views)
- Use **smart comments** (not spam) — better retention & ranking
- Ramp-up slowly — 100 → 500 → 2000 views/day max per IP pool
- Monitor **success rate** — below 70%? Switch proxies/fingerprints
- Avoid 24/7 — add natural sleep cycles (night pauses)
- Combine **views + likes + comments** for strongest ranking push
